# Onboarding Flow (POC)

## 1. Principles
- Frictionless signup: minimal required fields
- Immediate value: script evaluation in <60s
- Progressive disclosure: advanced options after first use

## 2. Steps
1. **Landing:** Clear value prop, email/social login
2. **Script Submission:** Paste/type script, select type, optional context
3. **Evaluation:** Instant feedback, actionable highlights
4. **Rewrite:** AI suggestions, accept/modify
5. **Dashboard:** Progress snapshot, learning recs

## 3. UX Inspirations
- Copy.ai: Role-based personalization, quick time-to-value
- Notion AI: Template-driven, adaptive onboarding
